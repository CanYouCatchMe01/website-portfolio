My contributions:
  • The collision system is based on segment vs box interaction.
  • The intro and outro scripted events.
Segment vs box collision system:

During the project I implemented the collision system which was based on javidx9 Youtube video.Â The system works by that dynamic object creates a segment from its current position to the next position in the direction of its velocity. If the segment hits a box it will move the dynamic object to the hit point. If the segment does not hit a box it will move the dynamic object to the end of the segment. This solution is frame independent so the dynamic object will still collide with boxes during very low framerates.
