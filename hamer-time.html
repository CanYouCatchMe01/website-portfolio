Hammer Time

Marcus Cazzola
February 4, 2023
Updated on February 4, 2023
Details:
  • A 2D game inspired by Ikaruga.
  • 8 week development (20h/week) with a team of 5 programmers, 3 level designers and 4 artists. Game project 3 at The Game Assembly.
  • Made in TGA2D, the school's in-house C++ game engine.
My contibutions:
  • Player movement.
  • Xbox and keyboard input manager.
  • Sprite sheet based animations.
Spritesheet based animations:

During the project, I had to develop a pipeline for importing sprite sheets into the TGA2D game engine. I came up with a solution where the artist used TexturePacker to export the sprite sheet and a JSON file that described the coordinates of each sprite. I then created a pipeline that could read the JSON file and develop an animation system capable of switching between different animations.
